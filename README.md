# Trabalho de Aplicações Web e Móveis

Este repositório contém o desenvolvimento do meu portfólio como parte dos estudos em desenvolvimento web. O objetivo deste projeto é praticar e demonstrar habilidades fundamentais na construção de interfaces modernas, organizadas e responsivas.

## 🚀 Sobre o projeto

O portfólio foi desenvolvido utilizando tecnologias básicas da web, com foco em estrutura, estilização e interatividade. Durante o desenvolvimento, busquei aplicar boas práticas de organização de código e responsividade para diferentes tamanhos de tela.

## 🧠 Aprendizados

Durante a criação deste projeto, adquiri conhecimentos importantes, como:

- Utilização do **Grid Layout** para organização de elementos na página
- Uso de **Flexbox** para alinhamento e distribuição de componentes
- Manipulação básica com **JavaScript (JS)** para interatividade
- Estruturação correta com **sintaxe HTML**

## 💻 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 📱 Responsividade

O projeto foi desenvolvido pensando em diferentes dispositivos, desde desktops até telas menores, buscando uma boa experiência de usuário.

🚧 Em desenvolvimento — melhorias e novas funcionalidades serão adicionadas com o tempo.
